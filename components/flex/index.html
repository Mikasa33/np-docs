弹性布局 Flex ​

一个用于弹性布局的占位小组件。

演示 ​

基础 ​

使用 NpFlex1 实现 css 中的 flex: 1 效果。

查看代码
vue
<script setup lang="ts">
import { NButton, NFlex } from 'naive-ui'
import { NpFlex1 } from '@rezero/np'
</script>

<template>
  <NFlex>
    <NButton>按钮1</NButton>
    <NpFlex1 class="bg-#eff0f3 dark:bg-#282a2f" />
    <NButton>按钮2</NButton>
  </NFlex>
</template>
